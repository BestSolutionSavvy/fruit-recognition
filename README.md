# Fruit Recognition

> ⚠️ Placeholder repository
> The entire project was developed in Google Colab, therefore this repository currently only contains project material and documentation.

## Project Description

The notebook aims to recognize fruits and vegetables in either a fresh or rotten state.

To achieve this goal, the project follows a feature extraction and comparison approach in order to train different classifiers based on the extracted features.

## Workflow

1. Image loading

2. Dataset splitting into training, validation, and test sets

3. Feature extraction

   * Color

     * 3D HSV Histogram
     * Color Moments
   * Texture

     * Gabor
   * Shape

     * Sobel

4. Search for the best classifiers

5. Classifier initialization

6. Training of three classifiers

7. Classifier performance analysis

8. Borda count

9. Classifier using all features

10. Performance analysis

11. Error analysis

12. Deep Learning comparison
